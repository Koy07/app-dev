##Teasing Master Takagi-san
- is a Japanese manga series written and illustrated by Sōichirō Yamamoto. It started in Shogakukan's shōnen manga magazine supplement Monthly Shōnen Sunday Mini in June 2013, and later moved to the main magazine, Monthly Shōnen Sunday, in July 2016, and finished in October 2023.

**Characters
- Takagi san - main protagonist, a girl who always like to tease his friend Nishikata and secretly has a crush on him
- Nishikata - main protagonist, a buy who is always teased by Takagi-san
- Yukari - supporting character
- Sanae - supporting character
- Mina - supporting character
- Takao - supporting character
- Kimura - supporting character
